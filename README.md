# ランダムデータ作成スクリプト

このPythonスクリプトは、ランダムな個人情報（氏名・カナ・年齢・性別・電話番号・生年月日・郵便番号・住所・メールアドレス）を自動生成し、CSV形式でデスクトップに保存するツールです。

## 📦 主な機能

- 氏名（ランダム）
- 氏名（カナ）
- 年齢（20～30歳）
- 性別（男性 / 女性）
- 電話番号（090 / 080から始まる番号）
- 生年月日（適当な年月日）
- 郵便番号（東京都のいくつか）
- 住所（郵便番号に応じた都内の住所）
- メールアドレス（ランダムなユーザー名＋ドメイン）
